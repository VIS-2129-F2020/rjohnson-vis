##Defining the data set

Below, I define a data set including the following variables:

  • HINCP - Household income - continuous
  • RNTP - Monthly rent ($) - continuous
  • NP - Number of people per household - continuous
  • BDSP - Number of bedrooms per household - continuous
  • HHT_label - Household type - categorical
  • BLDG_AGE_dec - Age of building in decades - categorical

This sample began with all Texas responses from the 2018 ACS 1-year survey, and was further narrowed to include only those households with some household income and monthly rent payment. Then, I added a filter to narrow my data to only those responses collected in Travis County.

Travis_puma_rentals <- get_pums(variables = c(
                                      "PUMA", 
                                      "HINCP", 
                                      "RNTP", 
                                      "NP", 
                                      "BDSP",
                                      "HHT",
                                      "YBL"),
                        state = "TX",
                        year = 2018, 
                        survey = "acs1",
                        recode = TRUE) %>%
     mutate(PUMA = as.numeric(PUMA)) %>%
     mutate(BLDG_AGE_dec = case_when(
      YBL_label == "1939 or earlier" ~ 8,
      YBL_label == "1940 to 1949" ~ 7,
      YBL_label == "1950 to 1959" ~ 6,
      YBL_label == "1960 to 1969" ~ 5,
      YBL_label == "1970 to 1979" ~ 4,
      YBL_label == "1980 to 1989" ~ 3,
      YBL_label == "1990 to 1999" ~ 2,
      YBL_label == "2000 to 2004" ~ 1,
      YBL_label == "2005" ~ 1,
      YBL_label == "2006" ~ 1,
      YBL_label == "2007" ~ 1,
      YBL_label == "2008" ~ 1,
      YBL_label == "2009" ~ 1,
      YBL_label == "2010" ~ 0,
      YBL_label == "2011" ~ 0,
      YBL_label == "2012" ~ 0,
      YBL_label == "2013" ~ 0,
      YBL_label == "2014" ~ 0,
      YBL_label == "2015" ~ 0,
      YBL_label == "2016" ~ 0,
      YBL_label == "2017" ~ 0,
      YBL_label == "2018" ~ 0,)) %>% 
     filter(HINCP > 0, RNTP > 0, PUMA > 05300 & PUMA < 05310) %>%
     select(HINCP, RNTP, NP, BDSP, HHT_label, BLDG_AGE_dec)
